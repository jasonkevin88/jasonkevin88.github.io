# 好游快爆AccessToken校验文档

  • 正式请求URL：
https://api.3839app.com/kuaibao/android/devsdk.php
1
  • 请求方式： POST

提交数据格式为JSON格式

  • 参数说明：
参数名 必选 类型 说明
c 是 string 控制器(authgame)
a 是 string 行为(checkAccessToken)
v 是 string 版本号(1546)
app_id 是 number 开发者游戏ID，快爆后台需要配置对应key
uid 是 number 登录用户的UID
access_token 是 string 登录授权码
  • 请求示例：
https://api.3839app.com/kuaibao/android/devsdk.php
{
  "c": "authgame",
  "a": "checkAccessToken",
  "v": "1546",
  "app_id": 5034,
  "uid": 726,
  "access_token":"129b7be6379c0edfa2fd1ec17dd04656"
}

参数名 类型 说明
code number 状态码

msg string 返回内容

返回相应的状态码说明

状态码 说明
100 判断成功
1000 判断失败
2002 参数错误
