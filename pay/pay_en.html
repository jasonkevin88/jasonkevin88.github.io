# 1.1 Integration

  1. DownloadSDKfirst, then copy common.aar login.aar anti.aar pay.aar alipaySdk.aar into your main module/libs.

Attention points：If there is access to the version before 1.1.0.0, please delete it firsthykb_sdk.aarandalipaySdk.aar

  1. Edit your main module build.gradle file，add the following code:
repositories {
  flatDir {
      dirs 'libs'
  }
}
dependencies {
  implementation(name:'common', ext:'aar')
  implementation(name:'login', ext:'aar')
  implementation(name:'anti', ext:'aar')
  implementation(name:'pay', ext:'aar')
  implementation(name:'alipaySdk', ext:'aar')
}

# 2. Development

# 2.1 SDK Initialization

Description： The initialization Api must be called before all other SDK related API calls.

Api:

//SDK Initialization API
HykbLogin.initSdk(Activity activity, String gameId, int orientation,  HykbInitListener listener)
1
2

Parameter decription：

Parameter Decription
activity current activity
gameId The game ID obtained after the game is created by the front desk of the fast explosive developer
listener HykbInitListener: Initialization Callback

Sample:

/**
* SDK Initialization
*/
HykbLogin.initSdk(MainActivity.this, GAME_ID, ActivityInfo.SCREEN_ORIENTATION_LANDSCAPE,new HykbInitListener() {

      @Override
      public void onInitSuccess(final HykbUser user) {
        if (user == null) {
          ToastUtil.showToast(MainActivity.this, "init success user = null ");
        } else {
          ToastUtil.showToast(MainActivity.this, "init success user = " + user.toString());
        }
      }

      @Override
      public void onInitError(final int code, final String msg) {
        HandlerUtils.runOnMainThread(new Runnable() {
          @Override
          public void run() {
            if (1102 == code) {
              //1102：Indicates the callback of exit game button in maintenance
              MainActivity.this.finish();
            } else if (2005 == code) {
              //2005：need exit game
              MainActivity.this.finish();
            } else {
              ToastUtil.showToast(MainActivity.this, msg);
            }
          }
        });

      }

      @Override
      public void onSwitchUser(boolean success, int code, HykbUser user) {
	    //The switch account here comes from the callback of the switch account logged in by cache
        if (success) {
          ToastUtil.showToast(MainActivity.this, "Account switch success user = " + user.toString());
        } else {
          ToastUtil.showToast(MainActivity.this, "Account not switched ");
        }
      }
    });

Class decription：

Initialize listening class： com.m3839.sdk.login.listener.HykbInitListener

Method Decription
onInitSuccess(HykbUser user) After the initialization is successful, call back the method. If the user is not empty, it means that the account is automatically logged

# 2.3 Login

Api:

/**
 * Login api call
 */
HykbLogin.login(Activity activity, HykbLoginListener listener);
1
2
3
4

Parameter Description：

Parameter Description
activity current activity
listener HykbLoginListener:Login callback

Sample:

    if (HykbLogin.getUser() != null) {
      ToastUtil.showToast(MainActivity.this, "Login success user = " + HykbLogin.getUser().toString());
      return;
    }
    HykbLogin.login(MainActivity.this, new HykbLoginListener() {
      @Override
      public void onLoginFinished(boolean success, int code, HykbUser user) {
        if (success) {
          ToastUtil.showToast(MainActivity.this, "Login success user = " + user.toString());
        } else if(code == 2005){
           MainActivity.this.finish();
        } else {
          ToastUtil.showToast(MainActivity.this, "Login failed code = " + code);
        }
      }
    });

# 2.7 Do Pay

Class HykbPayInfo： com.m3839.sdk.pay.bean.HykbPayInfo

Parameter Description
goodsName Product information
money Order amount
server server flag
cpOrderId The order number of the game
ext Additional fields

Api：

HykbPay.pay(Activivty activity, HykbPayInfo payInfo, HykbPayListener listener);
1

Sample:

    HykbPayInfo payInfo = new HykbPayInfo();
    Random r = new Random();
    payInfo.cpOrderId = "CXZ258654" + r.nextInt(1000);
    payInfo.money = 10;
    payInfo.goodsName = goodsName;
    HykbPay.pay(MainActivity.this, payInfo, new HykbPayListener() {
      @Override
      public void onPayResult(int code, String message, HykbPayInfo hykbPayInfo) {
        if (code == 9000) {
          ToastUtil.showToast(MainActivity.this, "Recharge success");
        } else if(code == 2005){
          MainActivity.this.finish();
        } else {
          ToastUtil.showToast(MainActivity.this, message);
        }
      }
    });

# 2.9 Description code：

code Description
-1 Server request exception
-2 Login call without initialization
-3 payment call without initialization
-4 Product name must be not empty
-5 The length of product name must be not more then 16 characters
-6 Money must greater than zero
-7 The maximum amount is not more than 50000 yuan
-8 The cpOrderId must be not empty
-9 The length of cpOrderId must be not more than 128
1101 Game not exist，or parameter not approved，please contact the operator for handling
1102 Game need: exit or logout
2001 User authentication failed
2002 Login failed
2003 Cancel login
2004 Login exception
2005 Game need: exit or logout
2006 Developer do re-login
2101 Payment not configured，lease contact the operator for handling
2102 Pay maintenance
2103 CP Order number error
2104 Order not exist
2105 Pay failed
2160 Pay success，white list test return
2161 Pay failed，white list test return
2199 Not found order
3001 Pay dialog close
3002 Pay limit dialog close
3004 Pay retry dialog close
4000 Server exception
4001 Data format error
6001 Pay Cancel
8000 Pay result confirming
9000 Pay success

# 2.10 Proguard

If you are using Proguard, add the following rules to your ProGuard configuration file:

# 不能混淆监听
-keep class com.m3839.sdk.common.**{*;}
-keep class com.m3839.sdk.anti.**{*;}
-keep class com.m3839.sdk.user.**{*;}
-keep class com.m3839.sdk.login.**{*;}
-keep class com.m3839.sdk.pay.**{*;}
-keep class com.alipay.sdk.app.PayTask{ public *;}
-keep class org.json.alipay.** { *; }
-keepclasseswithmembernames class com.m3839.sdk.common.js.JsInterface {
    <methods>;
}

# Prevent R8 from leaving Data object members always null
-keepclassmembers,allowobfuscation class * {
  @com.google.gson.annotations.SerializedName <fields>;
}
