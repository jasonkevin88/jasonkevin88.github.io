# SDK gamer verification authorization code

  • Official request URL:
https://api.3839app.com/kuaibao/android/devsdk.php
1
  • Request method: POST

Submit data in JSON format

  • Parameter Description:
Parameter name Mandatory type explain
c yes string Controller (authgame)
a yes string Behavior (checkaccesstoken)
v yes string Version number (1546)
app_id yes number The developer's game ID and the quick explosion background need to be configured with the corresponding key
uid yes number Uid of the login user
access_token yes string Login authorization code

-Request example:

https://api.3839app.com/kuaibao/android/devsdk.php
{
  "c": "authgame",
  "a": "checkAccessToken",
  "v": "1546",
  "app_id": 5034,
  "uid": 726,
  "access_token":"129b7be6379c0edfa2fd1ec17dd04656"
}
