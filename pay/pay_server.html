    # 好游快爆支付服务端接入文档

    # 游戏充值回调接口

    提供方：CP 调用方式：POST 使用场景：充值成功后，平台将通过服务端异步通知游戏方，发放相应的游戏币给玩家。

    • 输入参数
      orderId     订单ID
  gameId      游戏ID
  userId      用户ID
  money       支付金额
  gameMoney   游戏币
  server      区服
  cpOrderId   游戏方订单号
  time        时间戳，用于签名校验
  ext         游戏方下单提交的透传参数（原样返回）
  sign        签名，规则：1.key值字典序升序,2.按key1=value2key2=value2...key(n)=value(n)callbackKey拼接，其中callbackKey为对接时快爆生成的key。
