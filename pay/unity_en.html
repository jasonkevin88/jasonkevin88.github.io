      # HYKB payment SDK unity access document

      This document is a description of how unity engine games can be connected to the HYKB SDK The game engine used in the example is unity 2018.4.36f1. Pay attention to the compatibility of different versions

      Download the unity package example and extract it to get the following structure

      └── Assets
    ├── Plugins
    │   └── Android
    └── Scripts
        ├── com
        └── UnityPayDemo.cs

      Briefly describe the files in the assets directory

      1. Assets/Plugins/Android/**/*：The configuration file of gradle and the dependent package of SDK
      2. Assets/Scripts/com/**/*.cs：Encapsulation of java interface
      3. Assets/Scripts/UnityPayDemo.cs：Example of interface call

      # Import SDK

      Introduce dependency Copy the files in the assets directory to the corresponding location of the game project

      Assets/Plugins/Android//*.aar Assets/Scripts/com//*.cs
