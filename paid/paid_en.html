# 1. SDK接入

# 1.1 Android Studio Integration

  1. Download SDK first and then copy m3839_paid_checker.aar into your main module/libs.
  2. Edit your main module build.gradle file，add the following code:
repositories {
   flatDir {
      dirs 'libs'
   }
}

# 2.Development

# 2.1 Call Api

Description： the api of paid download check check with callback:

HykbPaidChecker.checkLicense(activity, appId, publicKey, listener);
1

Parameter description：

Parameter Description
activity current Activity
appId HYKB Platform paid download application ID
publicKey ras publick key from HYKB Platform
listener HykbCheckListener for sdk event callback

Sample:

String publicKey = "Applied public key";
String appId = "Applied appid";
HykbPaidChecker.checkLicense(activity, appId, publicKey,
        new HykbCheckListener() {
        @Override
        public void onAllowEnter() {

        }

        @Override
        public void onReject(int code, String errorMsg) {
          finish();
        }
     });
