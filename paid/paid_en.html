# 1. SDK接入

# 1.1 Android Studio Integration

  1. DownloadSDKfirst, then copy common.aar anti.aar paid.aar into your main module/libs.

Attention points：If there is access to the version before 1.1.0.0, please delete it first m3839_paid_checker.aar

  1. Edit your main module build.gradle file，add the following code:
repositories {
  flatDir {
      dirs 'libs'
  }
}
dependencies {
  implementation(name:'common', ext:'aar')
  implementation(name:'anti', ext:'aar')
  implementation(name:'paid', ext:'aar')
}

# 2.Development

# 2.1 Call Api

Description： the api of paid download check check with callback:

HykbPaidChecker.checkLicense(activity, appId, publicKey, listener);
1

Parameter description：

Parameter Description
activity current Activity
appId HYKB Platform paid download application ID
publicKey ras publick key from HYKB Platform
listener HykbCheckListener for sdk event callback

Sample:

String publicKey = "Applied public key";
String appId = "Applied appid";
HykbPaidChecker.checkLicense(activity, appId, publicKey,
        new HykbCheckListener() {
        @Override
        public void onAllowEnter() {

        }

        @Override
        public void onReject(int code, String errorMsg) {
          finish();
        }
     });

# 2.2 Proguard

If you are using Proguard, add the following rules to your ProGuard configuration file:

-keep class com.m3839.sdk.common.**{*;}
-keep class com.m3839.sdk.anti.**{*;}
-keep class com.m3839.sdk.user.**{*;}
-keep class com.m3839.sdk.paid.**{*;}
-keepclasseswithmembernames class com.m3839.sdk.common.js.JsInterface {
    <methods>;
}
