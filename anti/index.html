    # 1. SDK接入

    # 1.1 Android Studio集成方式（推荐）

    1. SDK下载完，将aar的 common.aar login.aar anti.aar single.aar 类库导入游戏工程libs目录下

    注意点：有接入1.1.0.0之前版本请先删除m3839_fcm.aar

    1. 在主工程的build.gradle文件下添加以下代码即可完成依赖，
    repositories {
   flatDir {
      dirs 'libs'
   }
}

    # 2. 代码接入

    # 2.1 sdk初始化

    描述：

    SDK的初始化方法

    接口方法:

    UnionFcmSDK.initSDK(Activity activity, UnionFcmParam param, UnionFcmListener listener);
    1

    参数定义：

    参数名称 描述
    activity 游戏主界面的Activity
    config UnionFcmParam 防沉迷的参数信息
    listener UnionFcmListener 检测回调监听

    示例代码：

        String gameId = gameIdEdit.getText().toString();
    UnionFcmParam param = new UnionFcmParam.Builder()
                .setGameId(gameId)
                .setOrientation(ActivityInfo.SCREEN_ORIENTATION_PORTRAIT)
                .build();
    UnionFcmSDK.initSDK(this, param, new UnionFcmListener() {
            @Override
            public void onFcm(int code, String message) {
                if(code == 100) {
                    //表示登录成功，获取登录信息
                    UnionFcmUser user = UnionFcmSDK.getUser();
                    if(user != null) {
                        Toast.makeText(SampleActivity.this, user.toString(), Toast.LENGTH_SHORT).show();
                    }
                }else if(2005 == code) {
                    finish();
                }else{
                    Toast.makeText(SampleActivity.this, message, Toast.LENGTH_SHORT).show();
                }
            }
        });

    类描述：

    防沉迷参数信息类：com.m3839.sdk.single.UnionFcmParam.Builder

    方法名称 描述
    setGameId(String gameId) 设置游戏ID，由快爆开发者平台上申请的参数
    setOrientation(int orientation) 设置游戏屏幕方向

    检测监听类： com.m3839.sdk.single.UnionFcmListener

    方法名称 描述
    onFcm(int code, String msg) 检测成功后回调方法

    常见回调的code描述：

    code 描述
    -1 服务端请求异常
    100 表示登录成功，根据个code，去获取用户信息UnionFcmSDK.getUser()
    1001 青少年模式已关闭
    1002 青少年模式已开启
    1003 未设置找回密码联系方式
    1004 青少年模式，“我知道了”的回调
    1101 游戏不存在，未进行审核，联系运营人员处理
    1102 游戏开发者需要，退出游戏操作或者注销账号
    2001 用户认证失败
    2002 登录失败
    2003 取消登录
    2004 登录异常
    2005 游戏开发者需要，退出游戏操作或者注销账号

    获取用户信息：

    UnionFcmUser user = UnionFcmSDK.getUser()
    1

    类描述：

    用户数据实体类： com.m3839.sdk.single.UnionFcmUser

    方法名称 描述
    getUserId() 获取用户uid
    getNick() 获取用户昵称
    getType() 获取登录方式
    getAccessToken() 这个字段用来登录授权的有效性校验，accessToken有效期两小时

    日志开关:

    UnionFcmSDK.setDebug(true);//true：open log output
    1

    释放资源:

    UnionFcmSDK.releaseSDK();
    1

    # 3. 混淆规则

    如果您需要使用proguard混淆代码，需确保不要混淆SDK的代码。 请在proguard.cfg文件(或其它混淆文件)尾部添加如下配置:

    -keep class com.m3839.sdk.common.**{*;}
-keep class com.m3839.sdk.anti.**{*;}
-keep class com.m3839.sdk.user.**{*;}
-keep class com.m3839.sdk.login.**{*;}
-keep class com.m3839.sdk.single.**{*;}
-keepclasseswithmembernames class com.m3839.sdk.common.js.JsInterface {
    <methods>;
}
