    # 1.1 Android Studio Integration

    1. DownloadSDKfirst, then copy common.aar login.aar anti.aar single.aar into your main module/libs.

    Attention points：If there is access to the version before 1.1.0.0, please delete it firstm3839_fcm.aar

    1. Edit your main module build.gradle file，add the following code:
    repositories {
  flatDir {
      dirs 'libs'
  }
}
dependencies {
  implementation(name:'common', ext:'aar')
  implementation(name:'login', ext:'aar')
  implementation(name:'anti', ext:'aar')
  implementation(name:'single', ext:'aar')
}

    # 2.Development

    # 2.1 Call Api

    Description： the api of paid download check Initialization:

    UnionFcmSDK.initSDK(Activity activity, UnionFcmParam param, UnionFcmListener listener);
    1

    Parameter Description：

    Paramter Description
    activity Current Game Main Activity
    param Anti - addiction configuration information
    listener UnionFcmListener for sdk event callback

    Sample:

     String gameId = gameIdEdit.getText().toString();
    UnionFcmParam param = new UnionFcmParam.Builder()
                .setGameId(gameId)
                .setOrientation(ActivityInfo.SCREEN_ORIENTATION_PORTRAIT)
                .build();
    UnionFcmSDK.initSDK(this, param, new UnionFcmListener() {
            @Override
            public void onFcm(int code, String message) {
                if(code == 100) {
                    // indicate login success
                    UnionFcmUser user = UnionFcmSDK.getUser();
                    if(user != null) {
                        Toast.makeText(SampleActivity.this, user.toString(), Toast.LENGTH_SHORT).show();
                    }
                }else if(2005 == code) {
                    finish();
                }else{
                    Toast.makeText(SampleActivity.this, message, Toast.LENGTH_SHORT).show();
                }
            }
        });

    Class Description：

    Anti-addiction Configuration Class：com.m3839.sdk.single.UnionFcmParam.Builder

    Method Description
    setGameId(String gameId) The game id apply from the Hykb developer platform ，must be set
    setOrientation(int orientation) Set game screen orientation

    Hykb Anti-addiction Listener Class：com.m3839.sdk.single.UnionFcmListener

    Method Description
    onFcm(int code, String msg) event callback

    Common code description：

    code description：
    -1 Server exception
    100 Indicates that the login is successful. Get the user(UnionFcmSDK.getUser()) information according to a code
    1001 Teen mode is off
    1002 Teen mode is on
    1003 Password retrieval contact information is not set
    1101 The game does not exist and has not been reviewed. Contact the operator for handling
    1102 Game developers need to exit the game operation
    2001 User authentication failed
    2002 Login failed
    2003 Login cancel
    2004 Login exception
    2005 Game developers need to exit the game operation

    Get user information：

    UnionFcmUser user = UnionFcmSDK.getUser()
    1

    Class Description：

    User data entity class： com.m3839.sdk.single.UnionFcmUser

    Method Description
    getUserId() get user id
    getNick() get user nick
    getType() get user type
    getAccessToken() This field is used to verify the validity of login authorization,is valid for two hours

    Log switch:

    UnionFcmSDK.setDebug(true);//true：open log output
    1

    Release resources:

    UnionFcmSDK.releaseSDK();
    1

    # 3. Progurad rule

    If you are using Proguard, add the following rules to your ProGuard configuration file:

    -keep class com.m3839.sdk.common.**{*;}
-keep class com.m3839.sdk.anti.**{*;}
-keep class com.m3839.sdk.user.**{*;}
-keep class com.m3839.sdk.login.**{*;}
-keep class com.m3839.sdk.single.**{*;}
-keepclasseswithmembernames class com.m3839.sdk.common.js.JsInterface {
    <methods>;
