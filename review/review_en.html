# 1.1 Integration

Download SDK first, then copy common.aar and review.aar into your main module/libs.

Edit your main module build.gradle file，add the following code:

repositories {
  flatDir {
      dirs 'libs'
  }
}
dependencies {
  implementation fileTree(dir: 'libs', include: ['*.jar','*.aar'])
}

# 2.3 Login

Api:

/**
 * Login api call
 */
ReviewSDK.login(Activity activity, LoginListener listener);
1
2
3
4

Parameter Description：

Parameter Description
activity current activity
listener LoginListener:Login callback

Demo:

    public void onLoginClick(View view) {

        if (!hasInit) {
            ToastUtil.showToast(ReviewActivity.this, "Initialization first");
            return;
        }

        if (ReviewSDK.getUser() != null) {
            ToastUtil.showToast(ReviewActivity.this, "Login success user = " + ReviewSDK.getUser().toString());
            return;
        }

        ReviewSDK.login(ReviewActivity.this, new LoginListener() {
            @Override
            public void onLoginFinished(User user, int code, String message) {
                if (user != null) {
                    ToastUtil.showToast(ReviewActivity.this, "Login success user = " + user.toString());
                } else if (code == 2005) {
                    ReviewActivity.this.finish();
                } else {
                    ToastUtil.showToast(ReviewActivity.this, message);
                }
            }
        });
    }

# 2.4 Get user information

Api:

 User user = ReviewSDK.getUser();
1

Class Description：

Class HykbUser： com.m3839.sdk.review.bean.User

Method Description
getUserId() get user id
getNick() get user name
getType() get login type
getToken() get user token
getBirthday() get birthday information with real name authentication (yyyymmDD) eg: 20170107

# 2.7 Description code：

code Description
1000 Login Success
2003 Cancel login
2005 Game need: exit or logout
2008 Anti-addiction dialog close
