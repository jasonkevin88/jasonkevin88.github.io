    # 1. SDK嵌入

    将common.aar和review.aar文件copy到您的项目libs下。

    在工程build.gradle中添加如下配置：

    repositories {
  flatDir {
      dirs 'libs'
  }
}
dependencies {
  implementation fileTree(dir: 'libs', include: ['*.jar','*.aar'])
}

    # 2. 接入代码

    # 2.1 SDK初始化

    描述：

    初始化接口必须在所有SDK的其他相关api调用之前进行调用。

    接口方法:

    //SDK初始化接口
ReviewSDK.init(Activity activity, String gameId, int orientation)
    1
    2

    参数定义：

    参数名称 描述
    activity 当前游戏的主Activity
    gameId 联系快爆运营获取ID
    orientation sdk的屏幕方向

    示例代码：

    /**
* SDK初始化方法
*/
ReviewSDK.init(this, inputEdit.getText().toString(), getOrientation());

    # 2.2 日志开关

    方法:

    ReviewSDK.setDebug(boolean debug); 用来开启SDK的内部的相关日志输出

    # 2.3 登录

    接口方法:

    /**
 * 登录api的调用
 */
ReviewSDK.login(Activity activity, HykbLoginListener listener);
    1
    2
    3
    4

    参数定义：

    参数名称 描述
    activity 当前游戏主页面的Activity
    listener LoginListener 登录回调监听

    示例代码:

    ReviewSDK.login(ReviewActivity.this, new LoginListener() {
    @Override
    public void onLoginFinished(User user, int code, String message) {
        if (user != null) {
            ToastUtil.showToast(ReviewActivity.this, "登录成功 user = " + user.toString());
        } else if (code == 2005) {
            ReviewActivity.this.finish();
        } else {
            ToastUtil.showToast(ReviewActivity.this, message);
        }
     }
});

    # 2.4 获取用户信息

    方法:

     User user = ReviewSDK.getUser();
    1

    类描述：

    用户数据实体类： com.m3839.sdk.review.bean.User

    方法名称 描述
    getUserId() 获取用户uid
    getNick() 获取用户昵称
    getType() 获取登录方式
    getToken() 获取登录凭证
    getBirthday() 获取有实名认证的生日信息(年月日) eg:20170107

    # 2.5 注销

    方法:

    ReviewSDK.logout(Activity activity);
    1

    用来注销用户的登录信息

    # 2.6 游戏退出SDK释放

    方法: 在游戏退出的时候去调用

     ReviewSDK.releaseSDK();
    1

    # 2.7 回调code的说明：

    code 描述
    1000 登录成功
    2003 取消登录
    2005 游戏开发者需要，退出游戏操作或者注销账号
    2008 防沉迷弹窗的关闭

    # 2.8 代码混淆

    如果您需要使用proguard混淆代码，需确保不要混淆SDK的代码。 请在proguard.cfg文件(或其它混淆文件)尾部添加如下配置:

    -keep class com.m3839.sdk.common.** { *;}
-keep class com.m3839.sdk.review.** { *;}
