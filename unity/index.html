      # 好游快爆SDK-Unity接入文档

      English Document

      本文档是对 Unity 引擎游戏如何接入 好游快爆 SDK 的说明 所用示例使用的游戏引擎是 Unity 2018.4.36f1, 不同版本注意兼容问题

      下载 unity封装示例 ，解压得到如下结构

      └── Assets
    ├── Plugins
    │   └── Android
    └── Scripts
        ├── com
        └── sample
      1
      2
      3
      4
      5

      简单说明Assets目录下的文件

      1. Assets/Plugins/Android/**/*.aar：gradle的配置文件和sdk的依赖包
      2. Assets/Scripts/com/**/*.cs：对 java 接口的封装
      3. Assets/Scripts/sample/**/*.cs：接口调用示例

      # 引入 SDK

      引入依赖 将Assets目录下的文件拷贝到游戏项目对应位置

      1. Assets/Plugins/Android/**/*.aar
      2. Assets/Scripts/com/**/*.cs
